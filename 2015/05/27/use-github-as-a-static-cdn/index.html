把github作为静态资源cdn

  • 分类： 笔记 ·
  • 发表于2015-05-27 09:55:46

博客是用hexo搭建的，没有上传图片的功能，于是想把图片都放github上。需要图片的时候![Alt text](/path/to/img.jpg "Optional title")这样就可以插入了。
