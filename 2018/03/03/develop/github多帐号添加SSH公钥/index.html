github有两个账号，但是同一个ssh公钥只能添加给一个账号，需要再生成一个密匙后添加。步骤如下：

再添加一个ssh密匙

1、指定邮箱

2、提示输入生成位置和文件名，取个不同的名称。如: /Users/you/.ssh/id_rsa1

3、输入密码，不用设置，一路回车到完成，生成的文件为：/Users/you/.ssh/id_rsa1
